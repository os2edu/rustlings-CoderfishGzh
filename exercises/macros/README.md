# Macros

Rust的宏系统非常强大，但也很难包装您
走来走去。我们不会教您如何编写自己的功能齐全的
宏。相反，我们将向您展示如何使用和创建它们。

如果您想了解有关编写自己的宏的更多信息，
[macrokata]（https://github.com/tfpk/macrokata）项目具有类似的样式
练习对沙发，但都是关于学习编写宏的全部。
